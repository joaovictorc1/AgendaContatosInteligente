# Agenda de Contatos Inteligente
Projeto em desenvolvimento em Python para a matéria de Projeto Integrador.

## Funcionalidades
- Adicionar contatos
- Buscar contatos por nome ou telefone
- Remover contatos
- Armazenar contatos em arquivo JSON

## Como executar
1. Instale o Python 3.10+.
2. Clone este repositório
3. Execute o aplicativo:
python main_gui.py
